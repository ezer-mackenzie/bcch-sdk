# Security Policy

## Reporting a vulnerability

If you discover a security issue in this project, please report it privately to the maintainer by email:

- ramirez.ruiz.eliezer.reuven@gmail.com

Please include enough detail for reproduction, including the affected version and a proof of concept if available.

## Public disclosure

Do not disclose security issues publicly before they have been addressed. This allows the project maintainers to respond and patch the issue.

## Supported versions

This project is under active development. Security fixes are accepted on the current development branch.

## Response

The maintainer will acknowledge the report and provide guidance for mitigation or patching.
