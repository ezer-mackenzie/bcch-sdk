# Banco Central Chile SDK

Bienvenido a la documentación del SDK para acceder al SieteRestWS del Banco Central de Chile.

Esta documentación cubre:

- Introducción y visión general del SDK.
- Guía de migración desde la implementación legacy (ver la [Guía de Migración](migration.md)).
- Referencia de la API pública del SDK (sync / async).
- Arquitectura interna y decisiones de diseño.
- Ejemplos de uso y patrones recomendados.

Nota: esta documentación está escrita en español. Si deseas una versión en inglés, puedo generar una copia traducida a continuación.

Comienza por leer la [Guía de Migración](migration.md) para entender las diferencias principales entre la implementación legacy y la nueva implementación alojada en `src/`.
