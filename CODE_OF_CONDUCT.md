# Code of Conduct

This project is dedicated to providing a welcoming and respectful community for all contributors.

## Our standards

All participants are expected to:

- Be respectful and considerate.
- Use inclusive language.
- Assume good faith in others.
- Provide constructive feedback.
- Avoid harassment, hate speech, and abusive behavior.

## Unacceptable behavior

Unacceptable behavior includes, but is not limited to:

- discrimination or harassment of any kind
- offensive language or imagery
- threats or bullying
- personal attacks
- deliberately derailing conversations

## Reporting

If you experience or witness unacceptable behavior, contact the project maintainer at:

- ramirez.ruiz.eliezer.reuven@gmail.com

## Enforcement

The project maintainers may take action to address behavior that violates this code of conduct. Actions may include asking for a change in behavior, closing issues, or blocking contributors from the project.
